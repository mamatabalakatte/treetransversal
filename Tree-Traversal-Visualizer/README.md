# Tree Traversal Visualizer

Live demo: https://mamatabalakatte.github.io/treetransversal/

A small browser-based visualization of binary tree traversal operations.

## What it does
- Insert, delete, and search values in a binary search tree
- Visualize preorder, inorder, and postorder traversal
- Show the tree structure interactively in the browser

## Files
- index.html: page layout and controls
- js/script.js: visualization logic
- cpp/: C++ implementation used for the tree logic
- wasm/: compiled WebAssembly assets

## Run locally
Open the project folder in a browser, or serve it with a simple static server.

Example:
```bash
cd Tree-Traversal-Visualizer
python3 -m http.server 8000
```

Then open http://localhost:8000/.
